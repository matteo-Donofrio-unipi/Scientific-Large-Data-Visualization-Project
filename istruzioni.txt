-----RENDERE ESEGUIBILE SU LINUX----------
...
...
path is :
progettoSlv/
	progetto/project folders & files (cartella progetto contiene il codice)
	venv/configuration files  (cartella venv contiene file config)
...
...

progettoSlv$ source venv/bin/activate (attiva ambiente virtuale)
progettoSlv$ export FLASK_APP=nome cartella progetto (associa la cartella con codice all'eseguibile, in questo caso il nome della cartella è "progetto")
progettoSlv$ export FLASK_ENV=development (setta la modalità dev che aggiorna l'eseguibile dopo ogni modifica)
progettoSlv$ flask run (attiva il local server sull'indirizzo locale http://127.0.0.1:5000/ )
